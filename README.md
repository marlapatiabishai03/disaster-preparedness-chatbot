📌 Project Overview

The Disaster Preparedness Chatbot is a full-stack web application designed to improve disaster awareness, preparedness, and emergency communication.

This platform enables students to learn disaster safety measures, attempt awareness quizzes, and receive real-time emergency alerts, while administrators manage content and monitor performance.

Built using secure backend architecture and structured database integration.


🧩 Tech Stack

Frontend: HTML5, CSS3, JavaScript  
Backend: Node.js, Express.js  
Database: MySQL  
Security: bcrypt, dotenv  
Version Control: Git & GitHub  

---

👤 User Roles

🎓 Student Panel
- Secure Login
- View Disaster Information
- Attempt Preparedness Quiz
- Track Latest Quiz Score
- Awareness Level Evaluation
- Receive Real-Time Emergency Alerts

🛠 Admin Panel
- Secure Login
- Add / Update / Delete Disaster Content
- View Student Quiz Performance
- Send Emergency Alerts

---

🔐 Security Implementation

- Passwords encrypted using bcrypt  
- Environment variables secured using dotenv  
- .env excluded from GitHub via .gitignore  
- No hardcoded database credentials  

---

🏗 System Architecture

Frontend → Express Backend → MySQL Database → AI Chatbot

⚙️ Installation & Setup (Run Locally)

1. Clone Repository:

git clone https://github.com/marlapatiabishai03/disaster-preparedness-chatbot.git

2. Install Dependencies:

npm install

3. Create .env file in root directory:

DB_USER=your_mysql_username  
DB_PASSWORD=your_mysql_password  
DB_NAME=disaster_chatbot  

4. Start Server:

npm start

Server runs at:

http://localhost:3000
